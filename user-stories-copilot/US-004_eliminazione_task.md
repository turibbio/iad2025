# US-004 — Eliminazione Task
Priorità: P0

Descrizione
- Come utente voglio eliminare una singola task per rimuovere attività non più necessarie.

Acceptance Criteria (Given / When / Then)
- Given che una task è presente nella lista,
  When l'utente clicca sul pulsante `Elimina (X)` associato alla task,
  Then la task viene rimossa dalla lista e la rimozione è persistita.

Note
- Considerare un comportamento di conferma opzionale (non obbligatorio per MVP).
