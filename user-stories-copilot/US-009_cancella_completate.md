# US-009 — Cancella Completate (Bulk Delete)
Priorità: P1

Descrizione
- Come utente voglio rimuovere tutte le task completate con una singola azione per pulire la lista.

Acceptance Criteria (Given / When / Then)
- Given che esistono task contrassegnate come completate,
  When l'utente clicca su `Cancella completate`,
  Then tutte le task completate vengono rimosse definitivamente e la vista/contatore si aggiornano.

Note
- Valutare eventuale conferma per evitare cancellazioni accidentali.
