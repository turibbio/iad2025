# Documentazione Frontend (React + TypeScript)

Questa sezione contiene la documentazione completa per lo sviluppo frontend con React 18 e TypeScript.

## Stack Tecnologico

- **Framework**: React 18+
- **Language**: TypeScript
- **Build Tool**: Vite
- **Styling**: CSS Modules o Tailwind CSS
- **State Management**: React Context API (app semplici) o Zustand
- **HTTP Client**: Axios
- **Form Validation**: React Hook Form + Zod
- **Testing**: Vitest, React Testing Library
