## Standard INVEST

Le user stories devono essere:

- **I**ndipendente
- **N**egoziabile
- **V**aluable
- **E**stimabile
- **S**mall
- **T**estabile

Genera le user stories in `USER_STORIES.md` con struttura organizzata per Epic.
