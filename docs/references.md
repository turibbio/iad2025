## Tools Consigliati

### IDE & Editors
- **Visual Studio 2022** - IDE completo per .NET
- **Visual Studio Code** - Editor leggero, ottimo per TypeScript/React
- **JetBrains Rider** - IDE cross-platform per .NET

### API Testing
- **Postman** - Client HTTP per testing API
- **Swagger UI** - Documentazione interattiva API (integrato in ASP.NET Core)
- **REST Client (VS Code)** - Extension per testing API

### Database
- **DB Browser for SQLite** - GUI per SQLite
- **SQL Server Management Studio** - Per SQL Server
- **Azure Data Studio** - Cross-platform per SQL Server

### Git Clients
- **Git CLI** - Command line (raccomandato)
- **GitKraken** - GUI cross-platform
- **SourceTree** - GUI gratuita

### Browser DevTools
- **Chrome DevTools** - Debugging web
- **React Developer Tools** - Extension Chrome/Firefox
- **Redux DevTools** - Se usi Redux

### Altri Tools
- **Docker Desktop** - Containerizzazione
- **Insomnia** - Alternativa a Postman
- **npm-check-updates** - Aggiornamento dipendenze npm
